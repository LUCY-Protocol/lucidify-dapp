## üì¶ Local Setup & Deployment

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/LUCY-Protocol/lucidify-dapp.git](https://github.com/LUCY-Protocol/lucidify-dapp.git)
    cd lucidify-dapp
    ```
2.  **Configuration:**
    Open `index.html` and update the following placeholders if deploying your own instance:
    - Contract Address (MATIC Tokenomics address)
    - ABI (If adding advanced contract interaction)
3.  **Run:**
    No compilation required. Simply open `index.html` in your favorite local server or Web3-enabled browser.
